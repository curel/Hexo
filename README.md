# 介绍
Hexo 本体备份。

## 常用命令

```bash
hexo clean // 清除缓存
hexo s // 生成本地服务器
hexo d -g // 打包上传 
```

## 新建文章

```bash
hexo new [post] // post: 标题名
```

## 阅读更多

在 `.md` 文本内插入 `<!--more-->` 即可生效。

## Typora

使用 Typora 编辑 `.md` 文档书写博文，Hexo 的标题、时间、分类以及标签的写法如下。

```markdown
---
title: 标题
date: 1999-1-8 00:00:00
categories:
- test
tags:
- test
---
```
